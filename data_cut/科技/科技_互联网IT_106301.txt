微软又出Win 10更新，这次却要讨好iPhone用户　　（原标题：微软又出 Win 10 更新，这次却要讨好 iPhone 用户）　　最近，关于微软Windows 10的一些数字让人瞠目结舌。新操作系统在推出两年后，装机量远超业界预期：日活跃用户数（每天使用超过3? 小时以上）跨越了3.5亿大坎，全球月活跃设备数则超过了5亿，据Gartner估计装机率会在今年达到85%；同时，Cortana的月活跃用户数也达到了1.4亿水平。　　今天，该公司又宣布了下一次Windows 10大版本更新—— Fall Creators Update（秋季创作者更新）的大量细节。　　首先，依旧免费。其次，这是一次主要讨好iPhone和Android用户的更新……　　你没看错……且听我慢慢解释。　　该更新中，将包含一个全新的，又有点似曾相识的新应用Story Remix。　　首先，Story Remix是一个相册应用，可以按照人物、地点、时间等维度来整理、保存和搜索你的所有照片和影像。似曾相识？因为和Google Photos的功能类似……Story Remix也是一个短视频制作工具，横跨Windows桌面和iOS、Android移动端，提供了一个非常简单的编辑界面，剪辑照片、影像并添加配乐，做成一段酷炫的小视频！　　Story Remix还有一些独特的小技巧，比如它的后端加入了图像、人脸识别和物体追踪技术。举个例子，你可以在一段动态影像里找到一个运动的物体（比如下图中的小孩），在他的身上加字幕，字幕就会跟他一起前进。如果影像里有好几个人，人脸识别可以找出所有的人，你可以在头像上选择一个人是“明星”，Story Remix就会自动生成以她为主角的短视频……　　更有趣的是，Story Remix可以和一个名叫Remix 3D的在线3D动态资源库连接（Remix 3D是微软去年宣布第一个Creators Update时推出的服务，被推迟到今年秋天才能发布。）Build现场演示了一个特别酷炫的画面：这是一个女子足球比赛的现场，一位球员准备发任意球。演示者在Story Remix里直接接入了Remix 3D资源库，调取了一个火球，然后选择“锚点到物体”，选择足球，然后演示者又调取了一个爆炸效果，锚点到球门。　　很快Story Remix就生成了新的视频：球员起脚抽射，足球变成了一个冲进球门，boom……　　一款轻量级的视频制作工具，功能却不只是简单的剪辑、配乐和生成。我发现Story Remix瞄准的其实是免费新手视频编辑软件这一市场，也就是Mac平台上备受欢迎的iMovie。其实这个市场Windows Movie Maker来的更早，但已经被搁置了相当长一段时间了，今年一月更是干脆终止了服务。　　所以你也可以说，这次微软带着Story Remix杀回来，打算挑战iMovie的地位，顺便“重新发明”Windows Movie Maker……　　角落里的Windows Phone用户提出了一个问题：有Windows Phone版吗？　　别担心，作为UWP应用，Story Remix肯定会支持Windows Phone的……　　Fall Creators Update的第二项新特性同样令人激动：全新的系统级设计语言Microsoft Fluent Design System。如果你是微软的忠实粉丝，可能早在去年年底听说过一个名叫Project Neon的新设计语言。Fluent Design和Neon是同一个东西，是后者的正式名称。　　你可能会感到混乱：微软又搞了一个新的设计语言？别担心，Fluent Design是微软在Win 7、Win 8时代的Metro/Modern UI的延续。你能在Fluent Design里看到经典的砖块状界面、按钮和熟悉的角落、侧翻浮现式动效，也能目睹一些新的，更加现代的界面呈现方式，比如banner栏元素尺寸随滚动动态变化，不同界面的层级区别等等。　　新设计语言Fluent Design在以下五个方面着重呈现：　　光影Light　　层次Depth　　动效Motion　　质感Material　　可延展性Scale　　漂亮是挺漂亮。看时间久了，我发现 Fluent Design倒是和iOS、Android现在各自的设计语言有很多相似的地方，比如毛玻璃效果，汉堡包菜单按钮和侧边栏等元素。其实这也是微软的用意所在，　　负责Windows开发的副总裁乔·贝奥福 (Joe Belfiore) 介绍，公司希望Fluent Design能够为用户创造出一种更美观、实用，更重要的是在Windows、iOS和Android等不同终端上跨平台统一的用户体验。　　当然，目前尚不清楚新设计语言会不会被移植到Windows Phone上……　　为什么很多办公人士都青睐苹果电脑，甚至iPad，而不是Windows PC？可能我们都经历过这样的场景，正在写一个文档或者PPT文件，因为软件崩溃系统死机，没有保存。不是说Mac不会死机，但至少几率小一些。　　而今天，微软推出了一个全新的，系统级的文件管理和历史内容追溯系统：Timeline（时间轴）。　　这个功能的入口在Windows 10下方工具栏上。点击多任务管理按钮，就可以看到当前，以及之前长达一个月时间里曾经开启过的任务，比如没有看完的网页、没来得及做完、保存的PPT文档等。　　Timeline的后端基于微软的智能云平台Microsoft Graph，以及一个名叫Project Rome的远程系统管理技术。任何接入了该平台的微软第一方应用或其他第三方应用，都可以调用Timeline功能，Microsoft Graph将为应用创建锚点记录，包括可搜索的文字标题、摘要，以及一张缩略图，制作成一张卡片。搜索入口在Timeline界面的右上角。　　Project Rome SDK有很多的用处，比如在Windows PC和iPhone之间同步未完待续的任务——没错，就像苹果自家的Handoff功能那样。　　比如我准备去吃午饭，等同事的时候在电脑上的Edge浏览器打开了一篇文章看，还没看完就得走了，于是我锁上电脑，拿着手机出门了。Cortana突然弹出了这样一条通知：　　这个功能名叫Pick up where you left off（更换设备，工作继续），字面意义就是“回到你之前离开的地方”。看微软的意思，这个同步功能肯定会支持微软的第一方应用，比如Office 365、OneNote，其他第三方软件支持还是要通过Project Rome SDK。而且根据我的了解，这个功能必须要登录微软账号才能实现……　　从桌面同步到手机的待续任务　　从手机同步到桌面的待续任务　　Project Rome SDK的跨平台同步技术，还有一个更直观的使用场景：云端剪贴板。我们可能都遇到过需要在电脑和手机之间传输内容，大多数人都用过微信PC客户端里的文件传输助手，也应该有人听说过Pushbullet这个东西。微软也想要打通系统层面，结束跨平台传内容的尴尬处境，于是在下一代Windows 10里直接做一个系统级的云端剪贴板功能Clipboard，可以在iPhone、Android手机和Windows PC之间来回复制图片、文字、链接、动图等。　　云存储服务 OneDrive 也是下次更新的重要内容。OneDrive 的一个核心功能是在硬盘上设立一个专区（文件夹），保存到里面的内容都可以被同步到云端，方便跨设备浏览使用。但是它的技术逻辑有时候让人感觉不舒服，比如这个专区里面所有的文件，都会在登录相同微软账号的机器上同步，无法单独选择同步或不同步的文件，对于那些存储量不足的用户而言，太占地方。　　新版 Windows 10 里的 OneDrive 同步功能，终于将要解决这个历时已久的问题。新的逻辑是文件随时同步随时取用，而且可以单独选择同步与否，不再强制放置于一个指定的文件夹。　　比如我可以在桌面创建一个文本文档，保存，右键点击文件，就可以自定义同步配置了。同样，在文件浏览器里，会有专门一栏显示文件状态，比如完全在云端、在本地还是两端都有，是否完成了同步等，消除了文件同步状态的混乱。　　总结完 Windows 10 Fall Creators Update 的新特性，相信大家都会对这次微软对 iOS 设备的宽容度留下深刻印象。这么想没毛病，毕竟微软的 Keynote 里放了这样一张图：　　仔细看，Windows Phone 在最边上……　　其实这是因为微软已经意识到，很多自己的用户同时也都是别家智能设备的用户。比如职场人士，手机用 iPhone，因为日常的办公软件限定平台，只能用 Windows 电脑。微软没必要为了自己平台的全方面发展，去给已经过气的小众平台（Windows Phone）太多的资源，反而因歧视 iOS、Android 而伤害了自己的用户。这次 Build 上，微软很清楚地释放出了对其他平台和公司的善意。　　微软全球执行副总裁，负责 Windows 和设备业务的特里·梅尔森 (Terry Myerson) 阐释了自己的想法：“今天，不同的设备和平台竞相争抢着我们有限的精力。我们会用平板看新闻、用手机查邮件，用电脑工作，每一种设备都满足我们多样化的需求……我们希望通过一个现代化的微软体系，帮助大家简化这个挑战。”　　微软的妥协让步，别的平台和公司看在眼里。其中一些公司决定采取同样的姿态。苹果就做了个好榜样：今天微软宣布，符合 Windows 应用设计标准的苹果 iTunes 将登陆 Windows 应用商店——这样，上月底 Windows 10S 发布时限定只能安装商城应用，不能安装 .exe 文件→没有 iTunes→iPhone 无法同步的问题就迎刃而解了。　　除了苹果，设计界和商业领域的知名软件公司 Autodesk 和 SAP 也都准备将旗下的旗舰产品，SketchBook 和 Digital Boardroom 放到 Windows 应用商城里。另外，微软还计划和 SUSE Linux、Ubuntu 以及 Fedora Linux 展开合作，允许用户直接在应用商城下载安装和使用这三款 Linux 的子系统。你应该可以说，Windows 10 马上就要成为最开放的桌面操作系统了……　　最后，我们什么时候能用上 Fall Creators Update 呢？　　对于 Windows Insider（内测用户社群）成员，今天就可以下载Insider Preview Build 16193 (PC)，包括以下特性：　　Story Remix 早期测试版　　UWP 应用单独调整音量　　对于非英文操作系统用户，恢复不会出 bug 了　　另外，微软还推出了一张新的忍者猫壁纸：
