　　腾讯体育讯-NHL季后赛东西部分区决赛全部结束。东部分区的匹兹堡企鹅最终通过双加时的抢七大战战胜参议院后，连续两年进入了总决赛。西部分区的掠夺者将黑八进行到底，淘汰小鸭后历史首次进入总决赛，腾讯体育将全程独家视频直播NHL斯坦利杯总决赛，喜欢冰球的球迷朋友们一定不要错过！　　2017年斯坦利杯决赛完整日程表（均为北京时间，客队在前）：　　Game 1：5月30日 周二（纳什维尔掠夺者vs匹兹堡企鹅）解说：柳亚鹏 孙大烨　　Game 2：6月1日 周四（纳什维尔掠夺者vs匹兹堡企鹅）解说：柳亚鹏 罗森　　Game 3：6月4日 周日（匹兹堡企鹅vs纳什维尔掠夺者）解说：孔祥宇 罗森　　Game 4：6月6日 周二（匹兹堡企鹅vs纳什维尔掠夺者）解说：柳亚鹏 孙大烨　　Game 5：6月9日 周五（纳什维尔掠夺者vs匹兹堡企鹅，如果需要）解说：孔祥宇 罗森　　Game 6：6月12日 周一（匹兹堡企鹅vs纳什维尔掠夺者，如果需要）解说：孔祥宇 孙大烨　　Game 7：6月15日 周四（纳什维尔掠夺者vs匹兹堡企鹅，如果需要）解说：孔祥宇 罗森
